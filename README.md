# NodeUpDL
Download NodeUp with Node!

### Install
```bash
$ npm install -g nodeupdl
```

### Use
To download the latest episode
```bash
$ nodeupdl latest
```
To list all episode titles
```bash
$ nodeupdl list
```
To download all episodes
```bash
$ nodeupdl all
```
To download a specific episode
```bash
$ nodeupdl 66
```
To download a range of episodes
```bash
$ nodeupdl 75-100
```
